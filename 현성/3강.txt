네 인물: 아브라함(12장-24장), 이삭(25장-26장), 야곱(27장-36장), 요셉(37장-50장)

3. 하나님과 고대 근동의 신들
하나님 - 유일하신 하나님 / 변함없고, 의로우심 / 전능하신 하나님 / 사랑의 하나님
고대근동의 신 - 다신주의 / 변덕스럽고, 비윤리적임 / 제한된 신 / 인간을 착취하는 신

4. 창조
태초에 하나님이 천지를 창조시니라

태초에 - 언제 - 우주의 시작
하나님이 - 누가 - 복수형 이름(성부, 성자, 성령)
천지를 - 무엇을 - 하늘들과 땅
창조하시니라 - 어떻게 - 질서의 창조(우주, 자연, 인간)

창세기 1장 2절 - 창조 전 상태
1. 혼돈 -> 질서정연  ㄱ
2. 공허 -> 채움        ㅣ-> 하나님의 말씀
3. 흑암-깊음 -> 빛    ㅣ

사람 - 육체: 흙, 영혼: 생기
사람만은 하나님의 형상을 닮도록 특별하게 창조

                                                 채우심
첫째날 - 빛과 어둠                             ->   넷째날 - 해, 달, 별
둘째날 - 궁창 아랫 물과 궁창 위의 물      ->   다섯째날 - 새와 물고기
셋째날 - 육지와 바다                          ->    여섯째날 - 땅의 생물과 사람


흙의 에너지 -> 식물이 가져감
식물의 에너지 -> 동물이 가져감
식물과 동물의 에너지 -> 사람이 가져감
사람의 에너지 -> 하나님께 영광(or 사탄이 가져감)

물질을 아는 지식 - 물리학, 자연과학
사람을 아는 지식 - 인문, 사회과학
학문 중 최고의 학문 - 철학
ㄴ> 영적 세계를 아는 지식 - 성경, 신학
철학은 질문하고 성경은 답한다

창세기 1장과 2장의 관계 (두 개의 창조기사)
서로 모순되지 않음
각기 다른 관점에서 상호보완의 역할
창1장: 질서의 관점
창2장: 관계에 초점을 맞춤

진화론은 사실이 아니다
워숭이가 사람이 되었다면 원숭이 피를 수혈 받을 수 있어야 하지만 그러지 못한다.
하등 동물이 고등 동물로 진화했다면 지금은 하등 동물이 없어야 하지만 아메바, 짚신벌레 등이 있다
(창 1:25)

폭발 전 최초의 원자 덩어리는 어디서 왔는가 (과학의 기본 원리인 인과 율의 원리에 맞지 않음)
(롬 1:20)


5. 창조의 하이라이트 - 인간창조
1) 특별한 존재(창1:26)
"우리" : 히브리인의 관점에서 이해

1. 장엄함을 나ㅏ내는 복수형
2. 천상의 존재들과 회의할 때
3. 히브리적 심사숙고의 복수형

2. 하나님의 형상
1) 조재론적: 피조물 중 최고의 존재
2) 사명적: 인간이 하나님을 대신하여 다스리는 왕의 기능
3) 관계적: 하나님과의 교제

창조의 원리
물고기 - 물속
나무 - 땅
새 - 날개 : 하늘
사람 - 하나님의 형상

6. 창조의 목적 - 안식
인간을 위해 6일 동안 우주만물을 창조, 일곱 째 날은 안식하심 (창 2:2-3)

7. 타락
1)타락의 이유 : 말씀의 변질 (말씀을 불순종)
창 2:17 - 선악을 알게 하는 나무의 열매는 먹지말라 먹는 날에는 반드시 죽으리라 하시니라
창 3:3 - 동산 중앙에 있는 나무의 열매는 하나님의 말씀에 너희는 먹지도 만지지도 말라 너희가 죽을까 하노라 하셨느니라

2) 선악과를 볼 때마다
1. 하나님의 명령을 기억
2. 하나님과 인간관계
3. 순종의 기쁨 - 에덴의 축복

3) 뱀을 통한 마귀의 유혹
죄와 벌 : 법을 어기면 죄가 되고, 죄를 지으면 벌이 있음 -> 죄를 지으면 반드시 죽음 -> 롬6:23
죄를 짓게 된 이유(사탄의 유혹) : 죄를 짓게 하는 자 사탄이 뱀으로 위장하여 아담과 하와를 유혹하고 결국은 죄를 짓게 됨 -> 창 3:4-5)

죄의 유전 : 부모와 자녀는 성과 모습과 성품이 똑같이 닮듯이 죄인의 자녀는 죄인이 됨 -> 롬 5:12
사형선고와 사형집행 : 죄로 말미암아 모든 사람이 사형선고를 받음. 사형선고를 받은 사람은 사형집행이 기다리고 있느 시한부 인생 -> 히 9:27

4) 관계파괴
1. 하나님과 인간의 관계
2. 인간과 인간의 관계
3. 인간과 자연의 관계

5) 첫번째 죽음 : 영적인 죽음(하나님과 분리되는 상태)
1. 소속과 신분이 마귀의 자녀
2. 영적인 문제로 시달림
3. 마음과 정신적 문제로 고생함

6) 두번째 죽음 : 욱체적 죽음(육체와 죽음이 분리되는 상태)
1. 모든 사람은 죽음 앞에서 두려워함
2. 죽은 사람들의 시신의 상태가 다름
3. 임사체험자들의 고백

7) 세번째 죽음 : 영원한 죽음(영혼이 천국과 지옥으로 분리되는 상태)
1. 심판이 있음 (히 9:27)
2. 불못에 던져지는 심판 (게 21:8)

8. 하나님의 은혜
1) 심판 중에도 사랑을 베푸심
2) 가죽옷을 지어 입혀주심
3) 원시복음 (창 3:15)

창조 -> 타락 -> 구속

2) 피조물 : 겸손
